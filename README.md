# Pacon

***
# Install:  
`cd /pacon ` 
`bash ./scripts/pacon_install.sh`    
`make   // build test tool`      
`cd ./pacon/server`  
`make` 

# Start Pacon  
`echo #NODE_IP > /pacon/config`  
`modify the file path in start_pacon.sh`  
`bash ./scripts/start_pacon.sh`  
`pacon   // run test`  

# Stop Pacon  
`bash ./stop_pacon.sh`
